# SanayiRehberim
MUG-Final-240053017

Araç sahiplerini sanayi ustalarıyla buluşturan Flutter mobil uygulaması.

## 🌟 Özellikler
* **Kullanıcı Paneli:** Dükkan arama, filtreleme, favoriye ekleme.
* **Randevu Sistemi:** Tarih ve saat seçerek randevu alma.
* **Bildirimler:** Randevu onay/ret durumunda anlık bildirim.
* **Yönetim Paneli:** Dükkan sahipleri için randevu yönetim ekranı.
* **Karanlık Mod:** Otomatik sistem teması algılama.
* **Harita:** Google Maps entegrasyonu ile yol tarifi.
